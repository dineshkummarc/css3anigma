Anigma
CSS3
While wondering in the woods Madelina's discovered a cave filled with jewels. Help Madelina collect all of the jewels in each cavern by touching jewels of the same color together.

Use the arrow keys or the mouse to select jewels and move them.

Hit 'E' to pull up the level editor and make your own levels.
